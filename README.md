# Employee-Input-System
Welcome to the Employee Input System! This system is designed to provide a comprehensive and efficient solution for managing employee information, as well as a platform for employees to provide feedback.

# Features
Add Employees: The system allows administrators to add new employees to the database. This feature requires the employee's personal information, including name, job title, department, and contact information.

Edit Employees: Administrators can also edit existing employee records, making changes to any of the information previously entered.

Delete Employees: Administrators can delete employee records, removing them from the database.

Employee Feedback: Employees can provide feedback through the system, either through a suggestion box or by completing a survey. The feedback will be directed to the appropriate department for review and action.

# Usage
Access the Employee Input System by visiting the designated URL.
If you are an administrator, log in using your administrative credentials.
If you are an employee, log in using your employee ID and password.
Administrators can add, edit, or delete employee records as needed.
Employees can provide feedback by clicking on the "Feedback" button on the main menu.

# Support
If you need assistance with using the Employee Input System, please contact your administrator or the IT department for support. They will be happy to help you with any issues or questions you may have.

Thank you for using the Employee Input System, and we hope that it helps you manage your employee information and feedback more efficiently!
